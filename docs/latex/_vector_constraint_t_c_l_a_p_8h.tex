1.\+0 Class that constraints the values of the vector to have a sum equal to (more or less) 1. We say more or less because we are dealing with doubles and we chose to give and Interval of validity of the given frequencies. If the the sum of the frequences is not one, or one frequence is negativ we will not accept this input and tell the user that there is a problem.
